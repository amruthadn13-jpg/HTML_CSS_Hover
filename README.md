# HTML_CSS_Hover

# 📚 Library Books UI (HTML + CSS)

A simple **Library Books webpage** built using **HTML and CSS**.
This project displays books in a **card layout using Flexbox** and includes a **hover effect on the borrow button**.

This project is part of my **web development learning journey**, where I am practicing how to design clean UI layouts using CSS.

---

## 🌐 Project Overview

This webpage shows a collection of books in a **modern card-style layout**.

Each book card displays:

* Book Title
* Author Name
* Borrow Button

The layout uses **CSS Flexbox** to arrange the book cards horizontally.

---

## 🖼 Project Preview

The webpage contains:

* A **header section** with the title *Library Books*
* **Book cards** for different books
* **Borrow button with hover effect**
* Clean spacing using **Flexbox layout**
